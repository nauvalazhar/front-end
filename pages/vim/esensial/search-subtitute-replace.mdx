# Search, Subtitute, dan Replace

Melakukan pencarian teks pada Vim begitu sederhana, kamu hanya perlu menggunakan tanda miring garis `/` dan disertai dengan *pattern* atau kata kunci yang kamu ingin cari.

```vi
/pattern
```

Misal, kamu ingin mencari bagian kode `userName`, kamu dapat menggunakan perintah `/userName`Â pada Vim.

Bila Vim sudah menemukan teks denganÂ *pattern*Â atau kata kunci yang kamu cari, kamu dapat menggunakan perintahÂ `n` yang berartiÂ *n*ext untuk melanjutkan pencarian dengan kata kunci yang sama. Sebaliknya, bila kamu ingin kembali ke teks yang ditemukan sebelumnya, kamu dapat menggunakan perintahÂ `N` (*ya, "N" kapital*) untuk kembali.

Seandainya Vim tidak menemukan teks denganÂ *pattern*Â atau kata kunci yang kamu sesuaikan, Vim akan mengeluh seperti ini:

```vi
E486: Pattern not found: pattern
```

PerintahÂ `/`Â membuat Vim mencari ke bawah dari posisi kursor saat ini. Bila sampai bawah tidak ditemukan, maka Vim akan mengulang pencarian kembali dari atas. Hal seperti ini ditandai dengan komentar dari Vim yang secara tekstual seperti ini:

```vi
search hit BOTTOM, continuing at TOP
```

Sekiranya kamu merasa kata kunci yang kamu cari berada di atas kursor saat ini, maka alih-alih menggunakan tanda garis miringÂ `/`, kamu dapat menggunakan tanda tanyaÂ `?` dan disertai dengan kata kunci yang kamu ingin cari.

Bila awalnyaÂ `/userName`, maka menjadiÂ `?userName`. Kebalikan dari tanda garis miring, dengan tanda tanya maka Vim akan mencari ke atas dari posisi kursor saat ini. Bila tidak ditemukan, maka akan mengulang pencarian dari bawah. Hal ini ditandai dengan komentar dari Vim yang secara tekstual seperti ini:

```vi
search hit TOP, continuing at BOTTOM
```

Perlu diingat, pencarian di Vim pada dasarnya menggunakanÂ *case-sensitive**,*Â jadi kamu perlu memperhatikan huruf kapital dan bukan. Seandainya kamu ingin menghindariÂ *case-sensitive*, kamu dapat menggunakan perintahÂ `:set ignorecase` atauÂ `:set ic` versi yang lebih ringkas.

Kamu juga dapat mengembalikan fiturÂ *case-sensitive*Â tersebut dengan perintahÂ `:set noignorecase` atauÂ `:set noic`.

Cara lain untuk menghindariÂ *case-sensitive*Â pada saat pencarian di Vim adalah dengan menambahkanÂ `\c` pada akhir perintah pencarian. Merujuk ke perintah pencarian sebelumnya, maka akan menjadiÂ `/username\c`.

Kamu juga dapat menggunakanÂ *regular expression*Â untuk kata kunci pencarian. Sebagai contoh, kamu ingin mencari tanda titik koma pada setiap akhir baris, kamu dapat menggunakanÂ `/;$`; ingin mencari teks yang mengandungÂ `.com`, kamu dapat menggunakanÂ `/.\.com`.

Tentu kamu dapat menemukan referensi mengenai pencarianÂ *regular expression*Â yang lebih lengkap pada sumber daya lain. Saya hanya memberikan beberapa contoh sajaÂ *barusan*.

Kita sudah belajar mencari teks di Vim, saatnya belajar mengganti kata kunci yang ditemukan dengan teks lain atau di Vim disebut juga denganÂ *subtitute*.

Untuk melakukan *search and replace*Â di Vim, kamu dapat menggunakan perintahÂ `:s` yang berartiÂ *subtitute*Â dan lebih lengkapnya seperti ini:

```vi

```

Pada dasarnya seperti itu saja. `keyword` adalahÂ *pattern*Â atau kata kunci yang hendak kamu cari, sedangkanÂ `replaceKeyword` adalah teks yang menggantikannya bilaÂ *pattern*Â atau kata kunci ditemukan.

Seandainya kamu memiliki teks seperti di bawah dan dengan kondisi posisi kursor berada di huruf "k" baris pertama:

```vi /k /1
k eyword keyword
keyword keyword
keyword keyword
```

Maka dengan perintah sebelumnya, teks yang kamu miliki akan menjadi seperti ini:

```vi
replaceKeyword keyword
keyword keyword
keyword keyword
```

Ya, hanya satu yang diganti, tahu kenapa? Karena Vim hanya akan men-*subtitue*Â atau me-*replace*Â teks yang pertama kali ditemui saja dan pada baris di mana posisi kursor saat itu berada.

Seandainya posisi kursor berada di baris kedua seperti ini:

```vi /k /1
keyword keyword
k eyword keyword
keyword keyword
```

Maka dengan perintah yang sama, hasilnya akan seperti ini:

```vi

replaceKeyword keyword
keyword keyword
```

Kamu dapat menambahkanÂ `/g`Â diakhir perintah sebelumnya untuk melakukanÂ *subtitue*Â secara global. Perintah sebelumnya menjadi seperti ini:

```vi

```

Dengan perintah tersebut, semuaÂ *pattern*Â atauÂ kata kunci yang ditemukan akan di-*subtitue*Â oleh Vim, tidak hanya satu.

```vi /k /1
k eyword keyword
keyword keyword
keyword keyword
```

Dengan perintah sebelumnya, maka hasilnya akan seperti ini:

```vi
replaceKeyword replaceKeyword
keyword keyword
keyword keyword
```

Kamu juga dapat menentukan jangkauanÂ *subtitue.*Â Misal, kamu ingin melakukanÂ *subtitute*Â dari baris pertama hingga baris kedua, maka perintah ini dapat digunakan:

```vi

```

Tidak peduli di mana posisi kursor kamu berada, Vim tetap akan melakukanÂ *subtitute*Â hanya pada baris nomor 1 hingga baris nomor 2. Kira-kira seperti ini hasilnya:


Pada Vim, hal ini dapat dilakukan dengan perintah yang mirip seperti sebelumnya. Hanya saja, angka 2 diganti dengan tanda dolarÂ `$`Â sebagai intepretasi baris akhir pada dokumen yang sedang dibuka.

Dengan perintah di atas, maka kamu sudah dapat menebak hasilnya. Ya, semuaÂ *pattern*Â di semua baris akan di-*subtitute*.

PerintahÂ `1,$` dapat disingkat dengan tanda persenÂ `%`. Menjadi seperti ini:

```vi

```

Sama seperti fitur pencarian Vim sebelumnya,Â *subtitute*Â juga menggunakanÂ *case-sensitive*.Â Kamu dapat menggunakan cara yang sama seperti di fitur pencarian untuk mengabaikanÂ *case-sensitive*.Â Tambahan, kamu dapat menggunakan perintahÂ `i` di akhir perintah untuk mengabaikanÂ *case-sensitive*Â pada fiturÂ *subtitute*.


Selain itu, pada bagianÂ *pattern*Â juga kamu dapat menulisÂ *regular expression*Â untuk pencarian yang lebih lanjut.
